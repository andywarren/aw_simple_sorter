AW Simple Sorter allows you to display a set of posts in a responsive portfolio type layout and sort them by category similar to jQuery Quicksand.

== Description ==

AW Simple Sorter will allow you to display a set of posts in a responsive portfolio type layout and sort them by category similar to the way jQuery Quicksand works. Usage instructions included on plugin documentation page.

== Installation ==

To install the plugin follow these instructions:  
  
1. Download the plugin and unzip it.  
2. Upload the folder aw_simple_sorter to your /wp-content/plugins/ directory.  
3. Activate the plugin from your WordPress admin panel.  
4. Installation finished.
